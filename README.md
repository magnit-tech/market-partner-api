# Magnit Market Partner API

Документация по API для партнёров Magnit Market.

API позволяет управлять товарами, ценами и остатками, а также собирать и отгружать заказы.

## Авторизация

Доступ к API осуществляется по ключу, передаваемому в заголовке `X-Api-Key`.
